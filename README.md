## Quick Start

### Installation

```bash
# Clone repository
git clone https://github.com/Devansh5150/Chawkidaar.git
cd Chawkidaar

# Activate virtual environment
.venv\Scripts\activate  # On Windows
source .venv/bin/activate  # On Linux/macOS

# Install dependencies in editable mode
pip install -e .
```

### Usage

```bash
chawkidaar --help
```
